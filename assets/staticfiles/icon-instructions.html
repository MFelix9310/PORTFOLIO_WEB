Instrucciones para crear iconos de LinkedIn y GitHub

Paso 1: Preparar las imágenes

Para que los iconos se vean correctamente en el diseño de tu portafolio, debes:

  1. Crear dos archivos de imagen: linkedin-orange.png y github-orange.png
  2. Cada imagen debe ser preferiblemente transparente (PNG) y de color blanco (o muy clara)
  3. El tamaño recomendado es de 25px x 25px (se pueden hacer un poco más grandes si es necesario)

Paso 2: Guardar en la ubicación correcta

Guarda las imágenes en: static/images/social/

Paso 3: Verificar el resultado

Una vez guardadas las imágenes, reinicia el servidor Django y abre tu página principal para ver los iconos.

Consejos para editar las imágenes:

  • Si utilizas herramientas como Photoshop o GIMP, asegúrate de que el fondo sea transparente
  • Si no tienes acceso a software de edición, puedes buscar iconos SVG en blanco de LinkedIn y GitHub y guardarlos como PNG
  • Una opción fácil es usar la biblioteca de iconos Simple Icons, que tiene todos los logos corporativos en formato SVG: https://simpleicons.org/
